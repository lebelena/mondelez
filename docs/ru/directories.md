# Справочники

В системе есть 6 справочников: бренды, компании, лимит сметы, пользователи, дистрибьюторы и товары.
При выборе необходимого справочника в [рабочей области](./how_to_start.md) отображается список тех элементов, которые уже существуют в системе и команда по добавлению новых элементов.  

## 1. Бренды
Можно добавить вручную

## 2. Компании
Можно добавить вручную

## 3. Лимит сметы
По заявке от клиента корректируется лимит по адресной программе для 1 сметы

## 4. Пользователи
Добавляются администратором вручную


## 5. Дистрибьюторы
Добавляются администратором вручную


## 6. Товары
Добавляются вручную
